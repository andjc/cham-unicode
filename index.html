Cham Unicode

GitHub Repository

Welcome to Cham Unicode.

The Cham Unicode repository brings together information and resources to facilitate the use of Cham script for Eastern and Western Cham using Unicode.

Fonts

Cham Unicode fonts (Opensource) for the Eastern and Western Cham scripts.

Keyboards

Keyboard layouts for Cham languages in Cambodia and Vietnam.

Locales

Locales for Eastern and Western Cham languages.

The main documents that specify the Cham script in Unicode are:

  • Unicode chart for the Cham block; and
  • Unicode standard: chapter 11

Documents with additional information include:

  • N3120 (L2/06-257): Proposal for encoding the Cham script in the BMP of the UCS
  • Scriptsource Unicode status page for Cham
