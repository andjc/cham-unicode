Cham Unicode Support

Ongoing development of ICT support for Eastern and Western Cham. The following resources will be developed:

  • Unicode fonts
  • Keyboard layouts
  • Collation (sorting)
  • Locale support
  • Transliteration tools
  • Encoding conversion tools
  • Cham Web Development.
